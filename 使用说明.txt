月度生活费账本 · 使用说明（Windows 11）
==========================================

这是一个本地记账小工具：界面在浏览器里，但程序完全跑在你自己的电脑上，
不联网、不上传、数据只存在这个文件夹里。一个人用，记每月生活费和各项支出。


一、第一次使用（只需做一次）
------------------------------------------

第 1 步：装 Python（如果电脑已经装过，跳过这步）
   - 打开网址 https://www.python.org/downloads/
   - 点黄色的 Download 按钮，下载 Python 3。
   - 运行安装程序，【最重要】：勾选窗口最下方的
        “Add python.exe to PATH”
     再点 Install Now。
   - 不勾这个的话后面会失败，切记。

第 2 步：把这个文件夹放到你想放的位置
   - 比如 D:\记账 或 桌面 都行。整个文件夹别拆散。

第 3 步：双击运行  启动账本.bat
   - 第一次会自动下载所需组件，约 1-3 分钟，黑窗口里会滚动文字，正常现象，等它。
   - 完成后会自动用浏览器打开账本页面。


二、以后每次使用
------------------------------------------

   双击  启动账本.bat  就行。
   它会弹出一个黑色窗口（这是账本的“发动机”，别关），并自动打开浏览器。

   用完想关闭账本：把那个标题带“账本服务”的黑色窗口关掉即可。
   （直接关浏览器只是关了页面，发动机还在跑；要彻底停就关黑窗口。）


三、账本怎么用（界面操作）
------------------------------------------

   左边是操作区，右边是图表和账目。

   1. 第一次先点左侧“新增月份”，开一个当前月份。
   2. 在“本月设置”里填“每月生活费”（比如 2000），
      再把每月固定支出一项项加进去（房租、话费、订阅等，可带类型）。
   3. 平时记账：在“新增记账”里填金额（必填）、选消费类型（必填）、
      内容可不填，点“添加到本月”。日期会自动取当天。
   4. 右边能看收入/支出/结余，和消费类型饼图（月度 / 季度可切换）。
   5. 账目可以“编辑 / 删除”。

   ★ 重要：所有改动是先暂存的，必须点右上角“写入”才真正保存到电脑。
     没点写入就关页面，这次的改动会丢。顶部出现“有未写入的改动”就提醒你该写入了。


四、数据 / 备份 / 换电脑
------------------------------------------

   - 你的账本数据存在文件夹里的  data\ledger.json
   - 每次“写入”前，程序会自动把上一版本备份到  data\backups\
   - 想换电脑：把整个文件夹（含 data）拷过去，对方装好 Python 后双击 bat 即可。
   - 想手动备份：直接复制 data 文件夹到网盘或U盘。


五、常见问题
------------------------------------------

Q：双击后黑窗口闪一下说没有 Python？
A：没装 Python 或装时没勾“Add to PATH”。重装一遍，记得勾那个选项。

Q：浏览器没自动打开？
A：手动打开任意浏览器，地址栏输入   http://127.0.0.1:5000   回车。

Q：提示端口被占用 / 打不开？
A：可能 5000 端口被别的程序占了。用记事本打开“启动账本.bat”，
   把里面两处的 5000 都改成 5001（或别的数字），保存后重双击；
   浏览器地址也相应改成 http://127.0.0.1:5001 。

Q：第一次启动时 Windows 防火墙弹窗问是否允许？
A：选“允许访问”即可（只是本机自己访问，不会对外）。

Q：想开机自动启动？
A：按 Win+R 输入  shell:startup  回车，把“启动账本.bat”的快捷方式
   放进打开的那个文件夹即可。（不放也没关系，用时双击就行。）
